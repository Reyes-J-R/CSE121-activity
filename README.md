## ✨ Features

### Core Sections
- **Header & Navigation**: Responsive navigation menu with mobile hamburger toggle
- **Home Section**: Welcome banner with hero content
- **About Section**: School description and information
- **Courses Section**: Grid layout displaying 6 course cards with descriptions
- **Contact Section**: Fully functional contact form with validation
- **Footer**: School information and copyright

### Interactive Features
- 🎯 **Mobile-Friendly Hamburger Menu**: Toggles navigation on small screens
- 🔘 **Welcome Button**: Click to display "Welcome to our school!" message
- 📝 **Form Validation**: 
  - Empty field checking
  - Email format validation
  - Real-time error messages
  - Success confirmation alert
  - Auto-clear form after submission
- 🎨 **Hover Effects**: Smooth transitions on buttons and links
- 📱 **Fully Responsive**: Works on desktop, tablet, and mobile devices
